# authentik documentation development

## Purpose

authentik's documentation is a site built from sources in the repository. To
preview a change, a contributor runs the docs server locally with live reload.
This skill covers starting that server, building the static output, and locating
the source files behind the docs, integration guides, and API reference.

## When to invoke

- "Start the docs server" / "preview the documentation locally."
- "I edited a docs page — how do I see it?"
- "Build the static documentation site."
- "Where does the source for this docs page live?"

Not this skill: the product web UI (frontend) or the backend that serves the API
the docs describe (backend).
